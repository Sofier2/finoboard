# FinoBoard

## ERP-система для управління зверненнями громадян та бюджетними процесами

FinoBoard — це веб-застосунок, розроблений для автоматизації процесу подачі, обробки та контролю громадянських звернень, а також управління бюджетними ініціативами. Система дозволяє користувачам створювати заявки, переглядати їх статус, брати участь у голосуванні, а адміністраторам — керувати всіма процесами через панель управління Django.



## Мета проєкту

Метою проєкту є створення цифрової системи, яка:
- забезпечує прозору обробку громадянських звернень;
- автоматизує роботу адміністрації;
- дозволяє громадянам подавати та відстежувати заявки;
- підтримує механізм голосування та пріоритизації ініціатив.



## Використані технології

- Python 3
- Django (backend)
- MySQL 8 (база даних)
- Docker / Docker Compose (контейнеризація)
- HTML / CSS (Django templates)

-

## Як запустити проєкт

### 1. Клонувати репозиторій: 

```bash
git clone https://github.com/Sofier2/finoboard
```

### 2. Запустити Docker: 

```bash
docker-compose up --build
```

### 3. Відкрити у браузері: http://localhost:8000

##  Створення адміністратора

Для входу в адмін-панель потрібно створити суперкористувача: 

```bash
docker compose exec web python manage.py createsuperuser
```

Далі ввести:

- username
- email
- password

##  База даних

- СУБД: MySQL 8
- База даних: finoboard
- Хост: db (Docker service name)
- Порт зовнішній: 3307
- Порт внутрішній: 3306


##  Міграції

Міграції виконуються автоматично при запуску контейнера.

Ручне виконання міграцій: docker compose exec web python manage.py migrate

##  Порти:
Django	8000
MySQL	3307 → 3306

##  Зупинка проєкту: 

```bash
docker-compose down
```

##  Функціонал системи

### Користувач:

- Реєстрація та авторизація
- Створення заявок
- Перегляд статусу заявок
- Голосування за ініціативи

### Адміністратор:

- Управління користувачами
- Модерація заявок
- Зміна статусів заявок
- Робота через Django Admin
